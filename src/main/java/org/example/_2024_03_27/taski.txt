Техническое задание:
1. Создание классов:

Класс Транспортное Средство:

Поля:
название (типа String)
скорость (типа int, км/ч)
Конструктор для всех полей.
Геттеры и сеттеры для каждого поля.
Метод toString() для вывода информации о транспортном средстве.

Класс Автомобиль:
Наследуется от Транспортное Средство.
Дополнительное поле тип двигателя (типа String, например, бензиновый, дизельный).
Конструктор для всех полей (включая поля базового класса).
Геттеры и сеттеры для нового поля.
Переопределение метода toString().

Класс Велосипед:
Наследуется от Транспортное Средство.
Дополнительное поле тип велосипеда (типа String, например, горный, шоссейный).
Конструктор для всех полей (включая поля базового класса).
Геттеры и сеттеры для нового поля.
Переопределение метода toString().

2. Обработка массивов:

В методе main создать массив Транспортных Средств,
включающий в себя объекты Автомобиль и Велосипед.
Используя простой цикл (например, for или foreach),
вывести информацию о каждом транспортном средстве в консоль.
