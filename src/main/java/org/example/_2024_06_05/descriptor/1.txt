Когда файл открывается в операционной системе,
ему присваивается некий идентификатор, называемый
дескриптором файла. Этот дескриптор используется для
взаимодействия с файлом на низком уровне, то есть для
выполнения простых операций чтения, записи и других базовых
операций с файлом.

Класс FileDescriptor в Java предоставляет способ работать с
такими дескрипторами файлов. Он является абстракцией над
низкоуровневыми деталями операционной системы и предоставляет
методы для выполнения этих базовых операций с файлом.
Используя класс FileDescriptor, можно осуществлять чтение и запись
данных в файлы на очень низком уровне,
непосредственно взаимодействуя
с дескриптором файла.
