Классы приложения:

Student - класс для представления студента.
Course - класс для представления курса.
StudentRepository - класс для управления данными студентов и курсов.
StudentService - класс, содержащий бизнес-логику приложения.

Методы чтения данных из файла в классе StudentRepository
предполагают, что строки в файлах разделены запятыми и каждая
строка представляет собой запись.
